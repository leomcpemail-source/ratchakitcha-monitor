# 🕐 คู่มือการคำนวณเวลา ICT ↔ UTC

## สูตรการแปลงเวลา

### ICT (เวลาไทย) → UTC
```
UTC = (ICT - 7) mod 24
```

### UTC → ICT (เวลาไทย)
```
ICT = (UTC + 7) mod 24
```

---

## ตารางแปลงเวลาที่ใช้บ่อย

| เวลาไทย (ICT) | cron UTC | คำอธิบาย |
|---------------|----------|----------|
| 06:00 น.      | `0 23 * * *`  | เช้ามาก (UTC วันก่อน) |
| 07:00 น.      | `0 0 * * *`   | เช้า |
| **08:00 น.**  | **`0 1 * * *`**  | **แนะนำ** - หลังราชกิจจาเผยแพร่ |
| 09:00 น.      | `0 2 * * *`   | สาย |
| 12:00 น.      | `0 5 * * *`   | เที่ยง |
| 18:00 น.      | `0 11 * * *`  | เย็น |
| 20:00 น.      | `0 13 * * *`  | ค่ำ |

---

## ตัวอย่างการตั้งหลายเวลา

### ตรวจสอบ 3 ครั้งต่อวัน
```yaml
on:
  schedule:
    - cron: '0 1 * * *'   # 08:00 ICT (เช้า)
    - cron: '0 5 * * *'   # 12:00 ICT (เที่ยง)
    - cron: '0 11 * * *'  # 18:00 ICT (เย็น)
```

### ทุกวันจันทร์-ศุกร์ เวลา 08:00 น.
```yaml
on:
  schedule:
    - cron: '0 1 * * 1-5'  # Mon-Fri, 08:00 ICT
```

### ทุก 6 ชั่วโมง
```yaml
on:
  schedule:
    - cron: '0 */6 * * *'  # Every 6 hours
```

2. ใส่ cron: `0 1 * * *`

4. บวก 7 ชั่วโมง = **08:00 ICT** ✅

## ⚠️ ข้อควรระวัง

### 1. GitHub Actions มีความล่าช้า 3-10 นาที
- ตั้ง `0 1 * * *` อาจทำงาน 01:03-01:10 UTC
- = 08:03-08:10 น. เวลาไทย (ปกติ)

### 2. ไทยไม่มี Daylight Saving Time
- ใช้ UTC+7 ตลอดปี
- ไม่ต้องปรับเวลาตามฤดูกาล

### 3. Free Tier Limitations
- ไม่แนะนำให้ตั้ง cron ทุก 5-10 นาที
- แนะนำ: 1-4 ครั้งต่อวัน

## 🐛 Debug เมื่อเวลาไม่ตรง

### ปัญหา: ตั้ง 08:00 แต่ได้ 14:31
```
14:31 ICT - 7 = 07:31 UTC
```
→ แสดงว่า workflow ตั้งเป็น `31 7 * * *`

**วิธีแก้:**
```yaml
# ❌ ผิด (ทำงาน 14:31 ICT)
- cron: '31 7 * * *'

# ✅ ถูก (ทำงาน 08:00 ICT)
- cron: '0 1 * * *'
```

### ตรวจสอบใน Workflow File
1. ดูไฟล์ `.github/workflows/check-ratchakitcha.yml`
2. ดู cron expression
3. คำนวณ: UTC + 7 = ICT

## 💡 เคล็ดลับ

### จำง่ายๆ
```
เวลาไทย - 7 = เวลา UTC ใน cron
```

**ตัวอย่าง:**
- 08:00 - 7 = 01:00 → `0 1 * * *` ✅
- 12:00 - 7 = 05:00 → `0 5 * * *` ✅
- 18:00 - 7 = 11:00 → `0 11 * * *` ✅

### กรณีพิเศษ (เวลาไทยน้อยกว่า 07:00)
```
05:00 - 7 = -2 → (-2 + 24) = 22:00 UTC (วันก่อน)
→ cron: '0 22 * * *'
```
