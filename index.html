กำลังเชื่อมต่อ GitHub...

🔑 ใส่ GitHub Personal Access Token

Token จะถูกเก็บไว้ใน browser (localStorage)
👁️ แสดง/ซ่อน Token
📝 วิธีสร้าง Token:
  1. ไปที่ GitHub Settings → Tokens
  2. คลิก Generate new token (classic)
  3. เลือก scope: repo
  4. Copy token มาวางด้านบน

🔑 การจัดการ Token

✅ เชื่อมต่อกับ GitHub สำเร็จ

🔍 คำค้นหา (Keywords)

คำแนะนำ: ระบุคำที่ต้องการค้นหา เช่น "น้ำมัน", "คลัง", "ปิโตรเลียม"

📅 วันที่ต้องการตรวจสอบ

เลือกวัน: เลือกวันที่ต้องการให้ระบบตรวจสอบ (สามารถเลือกหลายวัน)

⏰ เวลาที่ตรวจสอบ

สำคัญ: ใส่เวลาเป็นเวลาไทย (ICT) ระบบจะแปลงเป็น UTC อัตโนมัติ
